# Wayland Scanner (for) Odin

# NOTICE!!!
This program is **not at all** stress-tested  
I have only tested it on **wayland.xml** and **xdg-shell.xml**  
If you use other xml files and they fail, please make an issue and send the xml file to <nobodyspecial1553@proton.me>

### Build
`odin build . -o:speed -disable-assert`

### Usage
Type `wayland-scanner-odin -h` for options
